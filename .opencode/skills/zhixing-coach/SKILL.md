# 知行AI教练 (Zhixing AI Coach)

AI + 真人教练混合的个性化健身训练方案生成系统。已验证8个真实案例、7种会员类型。

## 产品哲学

不是卖训练计划，是解决"为什么还没动"。AI负责效率，教练负责专业。

## 会员分类引擎

输入：10维特征向量 → 输出7种类型

```
D1: 训练经验 (0-5)
D2: 训练频率 (每周天数)
D3: 器械条件 (0-4)
D4: 伤病约束 (0-6)
D5: 生活约束 (工作性质+班型+夜班频率+通勤) → V4结构化
D6: 心理状态 (0-5) → V4从问卷Q19f直接映射
D7: 训练经历 (中断原因+最长周期+教练经历) → V4扩展
D8: 特殊需求 (备赛/技能学习等)
D9: 女性专项 (V4新增：经期影响程度 0-3)
D10: 动机模块 (V4新增：最长坚持+期望见效+最低替代)
```

### 7种会员类型

| 类型 | 名称 | 核心特征 | 策略 |
|------|------|---------|------|
| TYPE_A | 渐进入门型 | 零基础 | 第一周不训练+兜底+PMS+粗粮 |
| TYPE_B | 重建信心型 | 停训信心消退 | 继续旅程叙事+双路径+夜班方案 |
| TYPE_C | 进阶突破型 | 中级+器械/伤病瓶颈 | DUP+RPE+器械突破+季度全休 |
| TYPE_D | 精细优化型 | 高级+专项需求 | 左右平衡+赛前减量+专项营养 |
| TYPE_E | 同行教练型 | 健身教练 | 专业框架+术语速查+示例FAQ |
| TYPE_F | 老会员进阶型 | 已有信任关系 | 线性周期+上下肢分化 |
| TYPE_G | 高压工作型 | 工作压力大+内疚 | ABC三方案+成绩单封面+离别关怀 |

## 12个策略模块

| 模块ID | 触发条件 | 内容 |
|--------|---------|------|
| MOD_ABC | 高压工作 | 完整版/精简版/在家版三级方案 |
| MOD_SCORECARD | 有历史数据 | 先展示成绩再给建议 |

| MOD_NOBLAME | 有挫折记录 | 消除"丢脸感"话术 |

| MOD_DECISION_TREE | 中级+ | 加重/维持/退阶决策表 |
| MOD_MENTAL | 所有类型 | 状态差降级+心理策略 |
| **MOD_PMS** | **女性+经期影响** | **经期周期调整** |
| **MOD_HOUR** | **高峰时段** | **器械替换策略** |
| **MOD_COMMUTE** | **通勤≥60min** | **时间压缩策略** |

## 核心开发原则

1. 策略库持续进化 — 每服务一个新会员，引擎就更聪明
2. AI出框架+教练注入灵魂 — 不完全依赖AI，保留教练微调接口
3. 先小程序再App — 微信生态获客成本低
4. 完成比完美重要 — 小程序MVP先上线验证付费模型
5. 问卷越详细，结果越精准 — 10维输入驱动7种会员类型+12个策略模块
