# 知行AI教练 · 训练方案交付模板 V2

> 用途：AI生成原始方案后，按此结构整理成最终交付格式发给用户
> 交付方式：微信发送（建议用Canva/稿定设计做成图片格式，或直接发清晰排版后的文字）
> 原则：方案要看起来*值这个价*——排版整洁、信息完整、有专业感

---

**【学会自己做计划 · 实用框架】**

> 4周执行期结束后，你可以用这套方法自己设计下个周期的计划：

```
第1步：定分化方式（根据每周能练几天）
  ├─ 每周2练 → 全身训练×2
  ├─ 每周3练 → 全身训练×3（你现在用的）
  ├─ 每周4练 → 上下肢分化
  └─ 每周5-6练 → 推拉腿

第2步：每次训练选动作（按这个优先级）
  ① 主项复合 1-2个
  ② 辅助变式 1-2个
  ③ 孤立动作 1-2个
  ④ 核心动作 1个

第3步：定组数次数
  - 主项：4组 × 6-10次
  - 辅助：3组 × 10-12次
  - 孤立：3组 × 12-15次

第4步：每4周评估一次，用前面的判断规则调整
```

### 交付后的跟进

**首次发送后24小时**，发一条跟进消息：
> "XX，方案看了吗？第一次练完跟我说说感受，我会根据你的反馈帮你微调。"

**训练开始后第1周**，询问：
> "练了两次了，感觉怎么样？动作有不清楚的随时拍视频问我。"

## V2 相比 V1 新增/优化的模块

| 模块 | V1 | V2 |
|------|----|----|
| 设计思路说明 | 无 | 新增——让用户理解"为什么是这种方式" |
| 器械清单确认 | 无 | 新增——让用户再次确认，避免方案有误 |
| 有氧安排 | 简单 | 增加备用方案+强度选择理由说明 |
| 渐进负荷 | 只有加重 | 增加器械上限突破策略+用户自判断规则 |
| 建议购置清单 | 无 | **新增**——泡沫轴+弹力带，具体场景+价格 |
| 关节保护专项 | 无 | **新增**——🔴禁止/🟡谨慎/🟢安全/🟢康复/停止信号 |
| 营养建议 | 通用数字 | 改为按实际用餐场景+应酬应对策略 |
| 备用方案 | 无 | **新增**——精简版训练，给用户退路 |
| 自主训练框架 | 无 | **新增**——4步法，教用户自己做计划 |
| 泡沫轴融入热身冷身 | 无 | 明确列出泡沫轴在热身/冷身中的使用时机 |
| 4周复盘节点 | 无 | 新增——给教练和会员一个明确的复盘清单 |
