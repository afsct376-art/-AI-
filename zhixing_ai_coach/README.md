## 完整工作流程

```
用户付款 → 发问卷 → 拿数据
  ↓
00_技能-计划制定.md: 决策树分类 → 确定类型+策略模块
  ↓
01_Prompt模板.md: 填入结构化数据 → AI生成初版
  ↓
教练审核+质检(04_质检清单)
  ↓
生成Excel+说明书(遵循03_交付模板)
  ↓
交付+跟进+回写策略到00_技能
  ↓
策略库持续进化——来的人越多 引擎越聪明
```
