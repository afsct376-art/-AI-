# 知行AI教练 · 腾讯问卷核心字段设计 V3

> 用途：在腾讯问卷（wj.qq.com）中创建此问卷，用户付费后发送链接让其填写
> 建议名称：《AI健身教练 · 训练方案定制信息采集表》
> 完成时间：约12-15分钟（建议分两部分发送：基础信息+生活细节，减少一次填完的疲劳感）

| 设置项 | 建议值 |
|--------|--------|
| 问卷标题 | AI健身教练 · 训练方案定制信息采集表 |
| 问卷说明 | "你好！我是崔知行，8年健身教练。填完这份表后，我会结合AI和你给我的信息，出一份专属于你的定制训练方案。" |
| 预计完成时间 | 约8-10分钟 |
| 截止时间 | 不设置（长期有效） |
| 提交限制 | 每人限答1次（同一微信号/设备只能填一次） |
| 隐私设置 | 启用「匿名回答不可追踪」→ 但要求填写姓名用于方案 |
| 提交后提示 | "已收到！我会在24小时内把定制方案发给你。如果有紧急问题，直接微信联系我。" |
