## 三、每日必做：神经肌肉再教育 NMR（5分钟）

> 替代传统Cat-Cow热身。核心区别：不是"活动开"，而是"教大脑重新控制腰椎"。

### NMR流程（每次训练前执行，顺序不可颠倒）

```
┌─ 第一步：腹内压建立 ───────────────────────────────┐
│  90-90仰卧，手放腰部下方感受腰椎压向地面            │
│  吸气360度扩肋，呼气收紧腹壁，腰椎下沉               │
│  ⏱ 2分钟（约6-8次深呼吸）                          │
│  ✅ 标准：呼气时腰部与地面的压力均匀无缝隙           │
├─ 第二步：死虫式（慢速控制） ────────────────────────│
│  强调"先收紧核心再动四肢"的顺序                      │
│  腹壁绷紧 → 吐气出手出腿 → 控制回来                  │
│  ⏱ 1分钟（约4-6次/侧）                             │
│  ✅ 标准：全程腰贴地，无拱腰代偿                     │
├─ 第三步：四足位鸟狗式（慢速） ──────────────────────│
│  强调"先收紧核心再伸手伸腿"                          │
│  伸手伸腿时呼气，回来时吸气                          │
│  ⏱ 1分钟（约4-6次/侧）                             │
│  ✅ 标准：躯干不旋转，臀部不侧移                     │
├─ 第四步：站立位腰椎中立位感知 ──────────────────────│
│  手扶墙，做骨盆前后倾找到中立位                      │
│  中立位下做浅蹲（1/4蹲），感受腰椎不动的感觉         │
│  ⏱ 1分钟                                            │
│  ✅ 标准：浅蹲过程中腰椎角度不变化                   │
└─────────────────────────────────────────────────────┘
```

**为什么这比Cat-Cow有效**：周正权之前硬拉受伤，是**动作控制断裂**而非力量不足。NMR重建的是"大脑→腰椎"的神经通路，让他能在后续的训练中保持腰椎稳定。

## 四、每日反馈：腰椎评分系统（训练前填写）

### 评分标准

| 分数 | 状态 | 训练决策 |
|------|------|---------|
| **0分** | 完全无感 | 按计划训练 |
| **1-2分** | 轻微酸胀（正常训练反应） | 按计划训练 |
| **3分** | 明显酸胀但无锐痛 | 降低负荷20%，取消跳跃 |
| **4分** | 明显不适（走路有感觉） | 降低负荷20%，取消跳跃，NMR延长至8min |
| **5-6分** | 疼痛感（非放射） | 只做NMR + 核心训练 + 有氧，取消下肢力量 |
| **7-10分** | 放射痛/麻木/锐痛 | 立即停止训练，就医 |

> 每周日晚上对本周评分做回顾，连续3天评分≥4分 → 下周进入减载模式。

## 五、跳跃准入测试（第1周D1执行）

> 周正权90kg+腰突，胡慧泽能直接跳深（70kg+技术成熟）不代表他可以。**体重多20kg，腰椎冲击力多60-100kg。**

### 测试项目（全通过才允许进入跳跃训练）

```
□ ① 单腿站立平衡 ≥ 30秒（闭眼）
    → 测试本体感觉（腰突后本体感觉会下降）

□ ② 原地纵跳落地后保持腰椎中立位
    → 教练目测：落地时骨盆无前倾/侧倾/扭转
    → 允许的微小屈髋 ≠ 骨盆前倾

□ ③ 落地后无腰部不适
    → 主观感受，对比日常走路的感觉
```

### 未通过的处理路径

| 未通过项 | 替代训练（跳跃准入前） |
|---------|---------------------|
| ① 闭眼单腿站<30秒 | 每天额外3组单腿平衡训练 |
| ② 落地腰椎中立丢失 | 退回**原地提踵跳+落地hold**(脚不离地仅跟腱发力) |
| ③ 落地后不适 | 推迟跳跃，先用**药球下砸+弹力带侧滑**建立核心抗旋能力 |

> **预计**：正常1-2周内可通过，第3周应能进入正式跳跃渐进。

##### 第4周 = 减载 + 代谢重启（关键）

**训练减载**：
- 所有动作组数减半（3组→2组，4组→2组）
- 取消跳跃训练，改为弹簧跳/药球下砸
- 负荷降至第3周的70%
- 有氧减至20min

**代谢重启（营养）**：
- 热量回升至维持水平：**2800-3000kcal/天**
- 碳水恢复至3g/kg（~270g/天）
- 蛋白质维持200g/天
- 目的：重置瘦素水平，防止第5-8周代谢崩溃

**为什么需要代谢重启**：
> 持续热量缺口→甲状腺T3下降→皮质醇上升→基础代谢主动降低（代谢适应）
> 第4周减载时的热量回升 ≈ 告诉身体"没有饥荒，可以继续代谢"
> 这是第5-8周能继续减的关键，不是"浪费一周"

## 七、营养策略（分阶段细节）

### 第1-4周：保护期（蛋白质优先）

```
总热量: 2300-2400kcal/天
蛋白质: 200g/天  ← 保护腰椎韧带修复，这是硬指标
碳水:    3g/kg (~270g/天)
脂肪:    0.8g/kg (~70g/天)

碳水时机（关键）：
  训练前1小时:  40g碳水（如1根香蕉）
  训练后30min:  50g碳水+30g蛋白质（如米饭+鸡胸）
  目的: 避免低血糖影响动作控制（腰突患者最忌控制力下降）
```

### 第5-8周：攻坚期（碳水循环）

```
训练日（D1/D2/D4/D5）：
  碳水: 3g/kg (270g) / 蛋白质: 180g / 脂肪: 60g

休息日（D3/D6/D7）：
  碳水: 1.5g/kg (135g) / 蛋白质: 180g / 脂肪: 70g

高碳日（D2膝主导日，消耗最大）：
  碳水: 3.5g/kg (315g)

总热量: 训练日~2400kcal / 休息日~1900kcal
        平均~2150kcal（缺口约1200-1300）
```

### 水分与执行

| 项目 | 要求 |
|------|------|
| 饮水 | 3-4L/天（减脂期代谢废物需要水分排出）|
| 戒断 | 戒糖、油炸食品、过度加工食品 |
| 蛋白质来源 | 鸡胸/鱼/虾/鸡蛋/脱脂奶/豆腐 |
| 训练前加餐 | 训练前1h必须吃，不能空腹训练（腰突禁忌）|
