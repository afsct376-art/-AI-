## 四、周正权节奏密码（贯穿8周）

他之前的计划用三位数/四位数表示节奏（离心:暂停:向心），比如"222"=离心2秒、底部停2秒、向心2秒。

V3全程使用**他的语言**：

```
全写形式:   "8:0:0"     = 离心8个数，底部不暂停，爆发向心
           "3:0:0"     = 离心3秒，底部不暂停，爆发向心
           "6:0:0"     = 离心6个数，底部不暂停，爆发向心
           
动作备注语: "稳定下落，向心爆发" = 他以前的教练给他写的cue
           "底部停留2个数"      = 他听过的指令
           "爆发完成"          = 他听过的指令
           "组间休息3-6分钟"     = 他的休息模式
           "速度组组间休1分钟"   = 他做速度箱蹲时的规则
```

**阶段节奏表**：

| 阶段 | 下肢节奏 | 上肢节奏 | 跳跃节奏 |
|------|---------|---------|---------|
| 第1-2周 激活适应 | **6:0:0** 离心慢下，爆发上 | 2:0:1 | 无跳跃/准入训练 |
| 第3-4周 离心重建 | **4:0:0** 离心控制，向心爆发 | 2:0:1 | 落地控制+轻跳 |
| 第5-6周 力量积累 | **3:0:0** 开始过渡到向心优先 | 2:0:1 | PAP组间恢复 |
| 第7-8周 转化冲刺 | **0:0:0** 全程爆发完成 | 1:0:0 | 摸高冲刺+组间收腹跳 |

## 七、教练侧备注：这版计划的核心差异在哪

V3和市面上大多数康复计划最大的不同是——**我没让周正权"学新东西"**。

大多数康复计划的逻辑是"你现在受伤了，以前练的都不对，跟着我的新体系走"。但做过十年一线教练的都懂，一个已经练了4-5年的运动员，**他的神经肌肉适应路径已经固化**。让他学新动作的代价很高，尤其是在热量缺口和康复压力下。

V3的策略是：
- **康复底线内容**（NMR、腰椎评分、跳跃准入、减重营养）→ 新学的，必须做
- **力量训练内容** → 全部用他已经会了的格式（速度箱蹲、PAP组间跳跃、悬垂翻、L举腿、直臂下压、仰卧哑铃上拉、节奏语言）

这样他的大脑不需要同时处理"动作模式控制 + 腰椎保护 + 热量短缺"三件事——动作模式他闭着眼睛都能做，注意力全部留给腰椎保护和发力质量。

**最后一句**：这个计划他执行起来的第一感觉应该是"这练法我熟"而不是"这教练让我学好多新东西"。——"我熟"意味着他能坚持下来，坚持是康复里最重要的变量。
