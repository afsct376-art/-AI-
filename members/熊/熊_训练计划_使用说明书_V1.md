## 你完赛过铁三——这不是给普通人的方案

你能站在铁三的终点线那一刻哭了——那不是脆弱的眼泪，是Pro的勋章。游泳、骑行、跑步——你已经用身体证明了你的毅力。

但这次不一样：
- **90kg的体重**——每减1kg，越野跑时膝盖每步少承受约4kg冲击力
- **腰痛恢复期**——骑车被撞的后遗症，不是肌肉酸痛，是创伤恢复
- **50km越野跑备赛**——越野跑需要的力量类型和铁三不同：上坡推力 + 下坡控制 + 核心抗扭转

**三件事不矛盾——但需要先后顺序。这份方案的核心就是告诉你：什么优先，什么等待。**

## 三件事的优先级——不是同时做

| 优先级 | 目标 | 为什么 |
|--------|------|--------|
| **第一** | 腰痛恢复 | 没有健康的腰，越野跑 = 不可能。第1周的核心任务就是测试：哪些动作疼，哪些不疼 |
| **第二** | 减重 | 90kg/160cm——每减1kg = 膝盖每步少承受4kg。减10kg = 每步少40kg。比任何力量训练对越野跑的帮助都大 |
| **第三** | 越野跑专项 | 你的有氧基础不是问题。越野跑需要的力量和铁三不同——上坡推力、下坡控制、核心抗扭转 |

## 腰痛训练原则——每节课的必修

| 级别 | 动作 | 说明 |
|------|------|------|
| 🔴 禁止 | 传统硬拉、大重量深蹲、俯身划船、跳箱、负重跑步 | 对腰椎有冲击或大压力的动作——完全避免 |
| 🟡 谨慎 | 深蹲（从自重开始，腰不痛才加重量）、RDL（最轻哑铃，全程直背） | 腰有任何不适→立即停 |
| 🟢 安全 | 臀推/臀桥（零腰压）、死虫式、鸟狗式、高位下拉、坐姿划船 | 放心练 |
| 🟢 康复 | 死虫式+鸟狗式（每节课热身必做——核心稳定是保护腰椎的第一道防线） | 每天在家也可以做 |
| ⚠️ 停止信号 | 腰出现锐痛/刺痛/放射性疼痛→立即停，当天不练 | 酸胀/紧绷=正常，可以继续但减量 |

**第1周的核心任务不是训练——是测试。** 把你平时做的动作过一遍，哪些腰不疼→记录下来。疼的动作→跳过，第2周再试。

## 四周节奏

| 周 | 阶段 | 力量训练 | 有氧 | 腰的关键问题 |
|----|------|---------|------|------------|
| **1** | 恢复+测试 | 2次/周，轻量，核心+臀激活 | 游泳（腰痛期有氧首选）+ 低强度骑行 | 哪些动作疼？哪些不疼？ |
| **2** | 重建 | 3次/周，全身力量，逐步加量 | 跑步+骑行 维持 | 腰痛减轻→加重。不减轻→退回第1周 |
| **3** | 进阶 | 3次/周，上下肢+越野专项 | 跑步+越野模拟（爬坡走） | 上坡推力+下坡控制专项力量 |
| **4** | 减载+测试 | 2次/周，轻量，换动作 | 保持有氧，不冲PR | 测体重+腰围+越野跑5km计时 |

## 第1周——测试，不是训练

**这一周只做2次力量训练——不是偷懒，是测试。** 把下面这些动作过一遍，每个做完问自己：腰疼吗？

- 不疼 → 记录重量/次数 → 第2周加重
- 疼 → 跳过 → 第2周再试

**核心激活是每节课的必修：** 死虫式 + 鸟狗式 + 臀桥。这三个动作零腰压——腰痛期间的安全训练底线。

## 腰痛决策树——这是本方案最重要的工具

打开 Sheet **「渐进决策树」**，每次训练前看腰的状态，对号入座：

| 腰的感觉 | 今天做什么 |
|---------|----------|
| 完全无不适 | 正常训练，按计划走 |
| 轻微酸胀/紧绷 | 可以做核心+上肢+臀桥。深蹲和RDL跳过或用自重 |
| 动作中锐痛/刺痛 | 立即停。只做零腰压动作。持续疼→今天结束 |
| 早上起床就疼 | 只做上肢（下拉+划船+推肩）+死虫式。不碰下肢 |
| 连续3天不痛 | 可以尝试进阶。但一有不适→退回 |

## 训练结构

### 训练A（下肢+核心——每周1-2次）
- 死虫式 + 鸟狗式（必做）→ 猫式伸展 → 臀桥 → 深蹲测试 → RDL测试 → 高位下拉
- **越野跑迁移**：臀桥=上坡推力的来源。核心=下坡控制。单腿稳定=每步踩在不平地面的支撑。

### 训练B（上肢+稳定性——每周1次）
- 死虫式+鸟狗式（必做）→ 坐姿划船 → 高位下拉 → 推肩 → 面拉 → 平板支撑
- **越野跑迁移**：背阔肌和菱形肌=长途控臂和身体稳定的锚点。肩后束=长时间摆臂不酸。

### 训练C（越野跑专项——每周1次 替代一次有氧日）
- 跑步机爬坡走（模拟上坡）→ 倒退走+侧向走（模拟下坡）→ 单腿臀桥 → 农夫行走 → 可选HIIT
- **腰不痛才做。第1周跳过。第2周腰不疼开始加入。**

## 减重饮食——比力量训练更重要的4周任务

| 营养素 | 目标 | 怎么吃 |
|--------|------|--------|
| 总热量 | 1700-1900 kcal | 每周掉0.5-1kg。有氧消耗大——不要低于1500 |
| 蛋白质 | 120-140g | 每餐有蛋白质。训练后蛋白粉。减重过程中保护肌肉 |
| 碳水 | 180-220g | 三餐一拳。有氧日多20-30g——你是耐力运动员 碳水不是敌人 |
| 饮水 | 3L/天 | 腰痛期间多喝水——帮助组织修复 |

**核心原则**：先吃菜再吃肉最后吃饭——这个顺序血糖更稳 饱腹感更强 掉秤更快。每减1kg=越野跑时膝盖每步少4kg。减10kg=每步少40kg。

## 越野跑补给

- 训练<1h：正常吃，不需额外补给
- 训练1-2h：每小时运动饮料或能量胶
- 训练>2h/50km备赛：每小时30-60g碳水+电解质。赛后30min内：碳水+蛋白——恢复速度翻倍。

你完赛过铁三——赛中补给你有自己的习惯，保持就好。

## 经期前后

**经前一周**：体重涨1-3kg=正常水肿。腰更要注意——经前韧带松弛，更容易受伤。训练降强度。
**经期1-2天**：休息或游泳（低强度）。第3天恢复。

## 身体信号

| 信号 | 怎么办 |
|------|--------|
| 腰锐痛/刺痛 | 立即停。当天不练。连续2天还疼→看医生 |
| 腰酸胀但不疼 | 可以练但降级。死虫式+臀桥+高位下拉=安全 |
| 连续3天腰不痛 | 尝试下周进阶。一有不适→退回 |
| 训练+有氧后极度疲劳 | 减重+恢复伤病+训练——三线作战。这周多睡1小时 |
| 体重一周没掉 | 先看是不是经前。不是的话检查油脂和隐性碳水 |

## 50km越野跑备赛提醒

你完赛过铁三——身体的感觉你是知道的。50km越野跑的消耗≈铁三全过程，但地形更复杂。

- **赛前2周**：停止加重力量训练。越野跑训练减至平时60-70%
- **赛前1周**：力量训练减至1次（只做核心+上肢轻量）。越野跑只短距离轻松跑
- **赛前3天**：完全停力量，只拉伸+核心激活。碳水加载
- **比赛日**：注意下坡时核心收紧——腰痛期间下坡是风险点
- **赛后**：休息3-5天，散步+游泳。不碰器械。身体不是训练时变强——是恢复时

## 第4周后复盘

| 看什么 | 怎么评判 | 下周期 |
|--------|---------|--------|
| 腰痛恢复 | 比4周前减轻了多少？能做深蹲和RDL了吗？ | 完全消失=正式进阶。仍疼=继续康复 |
| 体重+腰围 | 掉了2-4kg？腰围小了？ | 这是4周最大的进步——比任何力量数字都重要 |
| 越野跑5km计时 | 比4周前快了吗？爬坡轻松了吗？ | 快了=力量和减重在起作用 |
