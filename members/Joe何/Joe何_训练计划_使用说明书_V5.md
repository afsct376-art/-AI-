## 拿到手第一件事

打开Excel，第一个Sheet叫 **「今天练什么」**——这就是你每次训练前要看的地方。

它会告诉你：
- 这周是第几周
- 今天是周一/三/五中的哪一个
- 今天的主题是什么（强度/容量/爆发/减载）
- 应该去哪个Sheet找详细动作

**然后往下翻，找到"RPE快速理解"——花30秒看懂它。** RPE是你接下来4周最重要的工具，比重量数字还重要。

---

## RPE是什么（用你的话讲）

RPE就是"你做完一组之后，凭感觉打个分"。

| 分数 | 感觉 | 用在哪天 |
|------|------|---------|
| 4-5分 | 非常轻松，做完怀疑"我今天练了什么？" | 减载周 |
| 5-6分 | 轻快有力，像弹簧 | 周五爆发日 |
| 6-7分 | 肌肉充血发胀，像充了气的气球 | 周三容量日 |
| 8-9分 | 很重，做完感觉"再来一个就起不来了"——但你没起不来 | 周一强度日 |
| 10分 | 拼了老命，完全力竭 | **本方案不用！** |

不用纠结精确到小数点。感觉对了就行。你当天睡没睡好、吃没吃够、压力大不大——都会影响实际表现。RPE让你跟身体对话，而不是死磕一个数字。

## 一周节奏速览

```
周一  强度日  重、稳、控
      └→ 打开Sheet「第1-3周训练」→ 找到"周一 强度日"
      └→ RPE 8-9，做完每组剩1-2次力竭
      └→ 训练后碳水吃够（蛋白粉+香蕉+牛奶）

周三  容量日  累、泵、充血
      └→ 打开同一个Sheet → 找到"周三 容量日"
      └→ RPE 6-7，组间短（45-60秒），高次数
      └→ 肌肉像充了气一样——享受这种感觉

周五  爆发日  快、轻、弹
      └→ 打开同一个Sheet → 找到"周五 爆发日"
      └→ RPE 5-6，追求速度不是重量
      └→ 如果觉得重——那就是太重了，减重

周六/周日  休息

（第4周是减载周，三天都做替换动作，后面会讲）
```

## 每次训练完整流程（55-65分钟）

### 训练前2小时：吃点东西

```
一根香蕉 + 一杯蛋白粉
或者：两片全麦吐司 + 2个蛋
```

不吃就练——训练质量打七折。不要空腹。

### Step 1：热身（8-10分钟）

打开Sheet **「有氧热身冷身营养」** ，从步骤1做到步骤8：

1. 原地踏步+手臂画圈（1分钟）
2. **靠墙静蹲**（30秒×2组）——膝盖不舒服时必做
3. 猫牛式（8次）
4. 肩关节环绕（前后各8次）
5. 摆腿前后+左右（各10次/侧）
6. 死虫式（每侧8次）
7. 弹力带肩胛激活（15次）
8. **神经激活（30秒，选一个）**：
   - 有低箱→跳箱5次（30-40cm高）
   - 都没有→原地高抬腿快速交替20秒
   - 周一强度日和周五爆发日必做这一步

### Step 2：力量训练（35-48分钟）

打开Sheet **「第1-3周训练」** ，找到今天对应的训练日。

每个动作后面有这几列要看：
- **重量**：参考值，根据自己水平调整
- **休息**：严格遵守——尤其容量日（45-60秒），别超时
- **RPE**：每做完一组，用感觉验证一下——太重就降，太轻下回调
- **降级**：如果今天状态差，按这列调整

### Step 3：有氧（10-14分钟）

力量训练后直接上跑步机：

| 周次 | 时长 | 坡度 | 感觉 |
|------|------|------|------|
| 第1周 | 12分钟 | 10% | 能断续说话 |
| 第2周 | 13分钟 | 10% | 同上 |
| 第3周 | 14分钟 | 12% | 稍累一点 |
| 第4周 | 10分钟 | 8% | 轻松——减载周 |

跑步机坏了？酒店楼梯爬楼梯，或者户外绕酒店快走。

### Step 4：训练后1小时内：吃东西

```
蛋白粉一勺 + 一根香蕉 + 牛奶
或者：一碗米饭 + 一块鸡胸肉
```

周一强度日碳水可以多加一根香蕉。训练后这一小时是蛋白质吸收效率最高的窗口——别浪费。

### Step 5：冷身（5分钟）

胸大肌拉伸 → 背阔肌拉伸 → 髋屈肌拉伸 → 腘绳肌拉伸 → 婴儿式 → 深呼吸5-8次。

别跳过。5分钟冷身换第二天少一半酸痛。

## 三个训练日详解

### 周一 · 强度日

**核心感觉**：重，但可控。做完一组感觉"再来一个就起不来了"——但你没有真的起不来。

**怎么判断重量对不对**：
- 目标RPE 8-9
- 如果你用18kg做完4组，感觉"还能做3-4个"（RPE 6-7）→ 太轻了，下次加
- 如果你用18kg做了2组，已经快力竭、动作变形 → 太重了，减到16kg

**训练后吃什么**：碳水比平时多——强度日神经系统消耗大，需要更快补糖原。

### 周三 · 容量日

**核心感觉**：肌肉充血发胀，像充了气。组间短（45-60秒），每一组都很赶——代谢压力是三天里最大的。

**注意**：容量日的重量比强度日轻，不要因为"轻"就擅自加重。容量日的挑战不在重量——在密度。短间歇+高次数=让肌肉在缺氧状态下做功。

**训练后**：碳水正常，蛋白质不变。

### 周五 · 爆发日

**核心感觉**：轻快有力。如果觉得重——就是太重了。

**爆发日最容易犯的错误**：用太重。50-60%极限重量，追求的是推/拉的速度，不是重量。爆发推起+慢离心3秒还原——"拉快放慢"。

**训练后**：不应该很累。如果周五练完很累，说明你用的重量太重了，下次减。

---

## 第4周 · 减载周

第4周完全不一样。原则叫 **"动但不同"**。

**不是"做轻一点"——是"做不同的动作"**。换动作=让主力关节换一个受力角度，等于给关节放假。

打开Sheet **「第4周-减载」** ，找到原来做的动作→换成右边的替换动作。

| 原来做的 | 第4周换成 |
|---------|---------|
| 哑铃卧推 | 俯卧撑（3秒下/爆发推） |
| 高位下拉 | 弹力带高位下拉（单臂） |
| 分腿蹲 | 高脚杯深蹲（轻哑铃） |
| 臀推 | 自重臀桥 |
| RDL | 早安式体前屈（轻量） |

**每个动作2-3组，RPE 4-5**。组间不赶时间。

**如果你没有弹力带**：替换表里每一行都有一列"如果没有"备选——没弹力带用最轻哑铃，没哑铃用自重。

**减载周结束时的理想感受**：精力充沛、关节轻松、"今天好像没怎么练"——这就是超量恢复即将发生的最佳信号。

## 第4周结束后

### 4周复盘（自己先看一遍）

1. **RPE数据**：强度日的RPE稳定在8-9了吗？容量日6-7？RPE比重量诚实
2. **力量变化**：三个主题日各涨了多少？哪个进步最明显？
3. **体重+腰围**：晨起空腹称重趋势，腰带松了几个孔？
4. **膝盖恢复**：分腿蹲的深度有没有加深？还疼吗？
5. **减载感受**：第4周结束是精力充沛还是仍然疲劳？
6. **哪个主题最适合你**：强度日/容量日/爆发日——哪个做得最舒服？

### 进入下个周期

第4周结束后→直接进入新的第1周。重量基于这个周期的数据上调——上肢+1-2.5kg，下肢+2.5-5kg。

**每3个月**：安排1周完全休息。不碰器械，只散步、拉伸、泡沫轴。这不是退步——是给身体做一次系统重置。休息完回来，力量不会掉，反而会涨。

## 遇到这些情况怎么办

### 今天状态极差（没睡好/压力大/感觉累）

**不要硬撑——降级。**

| 今天本来是 | 降级为 | 感觉 |
|-----------|--------|------|
| 强度日 | 容量日 | RPE从8-9降到6-7 |
| 容量日 | 爆发日 | RPE从6-7降到5-6 |
| 爆发日 | 休息或只热身+有氧 | 做个热身就是胜利 |
| 减载周 | 只做弹力带+拉伸 | 多滚泡沫轴 |

**定心丸**：一次降级不会影响长期进步。一次硬撑却可能受伤停练2周。这账你算得过来。

> 这个规则也写在封面页——每次打开Excel都能看到。

### 膝盖在分腿蹲时不舒服

**当场**：跳过今天的所有下肢动作，换成：
- 臀推/臀桥（3组×12-15次）
- 哑铃罗马尼亚硬拉（3组×10-12次）
- 站姿弹力带后踢腿（3组×15次/侧）

这三个动作膝盖几乎不参与——零膝压组合。

**长期**：如果连续2次训练膝盖都有不适→回到最浅深度重新开始，不急着加深。

### 哑铃20kg不够用了

7种突破策略在Sheet **「渐进与突破」** 里。最常用的三个：

1. **慢离心**：3-4秒慢慢下放 + 爆发推起——轻重量也能达到增肌效果
2. **单侧变式**：单腿臀推、单腿RDL、单臂划船——一边一边做，20kg够用
3. **1.5次法**：下放→起一半→再下放→完全起身 = 1次——单次双倍刺激

### 这周只能练2次

打开Sheet **「备用自主复盘」** ，用2练精简版。保留核心复合动作，压缩辅助。

### 高位下拉插片不够了

龙门架单臂下拉——20kg的重量单臂做，空间还很大。

## 日常饮食（不用称克数）

### 按周期吃的总方向

| | 训练周(1-3) | 减载周(4) |
|------|-----------|---------|
| 总热量 | 2100-2300 | 1800-2000 |
| 蛋白质 | 144-160g | 一样——蛋白质不减 |
| 碳水 | 230-260g | 160-180g——训练少了碳水也少 |

### 三餐实操

| 餐次 | 你现在的习惯 | 调整 |
|------|------------|------|
| 早餐 | 麦片+牛奶+1.5个蛋 | ✅ 保持！换2个整蛋，面包/馒头换全麦 |
| 午餐（食堂） | 围桌炒菜 | 米饭一拳大小，先夹肉和菜，油多的沥一下，七分饱停 |
| 晚餐（居家） | 跟家人吃 | 米饭减半或换杂粮，保证一份肉，蔬菜不限，油多先控油 |

### 训练前后的吃（重要！）

| 什么时候 | 吃什么 | 为什么 |
|---------|--------|--------|
| 练前2小时 | 香蕉+蛋白粉 或 吐司+蛋 | 提供能量，不吃训练质量打折 |
| 练中 | 水500-750ml，小口喝 | 维持表现 |
| 练后1小时内 | 蛋白粉+香蕉+牛奶 或 米饭+鸡胸肉 | 补糖原+启动修复，蛋白质吸收最快 |
| 周一强度日后 | 碳水多加一根香蕉 | 强度日神经系统消耗大 |

### 应酬（约每周2次）

- 当天午餐吃清淡/少吃
- 喝酒前吃蛋白质垫肚子（蛋/肉/豆腐）
- 能不喝就不喝——如果必须喝，白酒比啤酒好（啤酒是热量炸弹）
- 应酬后多喝水
- **一周21顿饭，2顿吃"不好"只占10%——不构成失败。**

## 建议买的两样东西（总共约80元）

| 东西 | 价格 | 干什么用 |
|------|------|---------|
| 泡沫轴 | 30-50元 | 训练前滚大腿（股四头肌+髂胫束）——尤其是膝盖不舒服的时候。训练后滚上背 |

放进行李箱不占地方，酒店和家里都能用。尤其弹力带——减载周没它就是没工具。

## 4周后你就能自己做计划了

这套框架记下来：

```
1. 每周能练几天？→ 定分化（2练=全身×2 / 3练=DUP / 4练=上下肢）
2. 每天什么主题？→ 强度日(重)→容量日(累)→爆发日(快) 轮换
3. 每4周一循环？→ 3周DUP + 1周减载（换动作）
4. 每3个月？→ 全休1周
5. 每个动作用什么重量？→ RPE说了算，不是数字说了算
```

掌握了这个节奏，你就不再需要一个教练告诉你"明天练什么"。你的身体和RPE数据会告诉你。

## 最后的定心丸

4周不长也不短。不长是因为刚好够看到变化和建立习惯；不短是因为需要你每周认真执行3次训练 + 管住7天饮食。

过程中有两件事你可能会遇到：

**第一，某周执行得不好。** 没关系。80%的执行率就是成功。不要因为一次没做好就全盘放弃。

**第二，某天状态差。** 翻开封面页看"状态极差降级表"，降一档训练。降级不丢人，硬撑受伤才亏。

4周后，不管数据怎么变，最重要的一件事是——你是不是已经有信心自己判断"下一次训练该怎么调"了。那个才是这四个星期最值钱的收获。
