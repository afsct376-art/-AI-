## V4 相比之前版本的核心升级

| 维度 | V3（线性周期） | V4（DUP波动周期） |
|------|-------------|----------------|
| 训练主题 | 每周单一主题（全周容量/全周强度） | **每天不同主题**（强度日→容量日→爆发日） |
| 重量选择 | 固定百分比（80% 1RM） | **RPE动态调节**（根据当天身体状态实时调整） |
| 减载周 | 做轻一点同一动作 | **换不同动作**（关节不同应力路径） |
| 热身 | 8步关节活动 | **新增神经激活动作**（第8步） |
| 营养 | 按周期调整总热量 | **新增训练前后窗口**（前2h+后1h） |
| 长期规划 | 4周循环 | **3小周期 + 1周全休**（每季度彻底重置） |

## 核心周期逻辑——先理解这个

```
一年的节奏：
  
  大周期（1年）
    身体重组，80kg  70-72kg，力量水平稳定提升
    
    中周期（1季度 = 3个月）
      小周期1  小周期2  小周期3  1周全休
                                    散步/游泳/拉伸/泡沫轴
      
      小周期（1个月 = 4周）
        第1周  第2周  第3周  第4周
        DUP    DUP    DUP    减载周
        递增   递增   冲击   换动作
        
        DUP的一周（每周3天，每天不同主题）：
          周一  强度日  80-85%，4-6次，RPE 8-9
          周三  容量日  65-70%，10-12次，RPE 6-7
          周五  爆发日  50-60%，3-5次爆发，RPE 5-6
```

**为什么DUP比线性周期更好？**
- 你已过新手期（1.5年训练基础），线性周期容易适应性停滞
- DUP每周内轮换3种刺激——神经系统始终面临"新"挑战
- 每次训练主题不同，有新鲜感，更容易坚持

## RPE怎么用（本方案的核心工具）

V4每个动作都标了目标RPE。这不是"建议"——是**第二重量选择标准**。

| RPE | 感觉 | 还剩几次力气 | 用在 |
|-----|------|------------|------|
| 4-5 | 非常轻松 | 8-10次+ | 减载周 |
| 5-6 | 轻松→中等 | 6-8次 | 爆发日 |
| 6-7 | 中等偏强 | 3-4次 | 容量日 |
| 8-9 | 很重 | 1-2次 | 强度日 |
| 10 | 极限力竭 | 0次 | 本方案不用！ |

**实战用法**：
- 强度日，计划写"单边18-20kg，RPE 8-9"
- 如果18kg做完4组，感觉"还能做3-4次"（RPE 6-7）→ 太轻了，下次加
- 如果18kg做完2组，已经快力竭（RPE 9-10）→ 减到16kg，保动作质量

**死磕数字 vs 跟随RPE**：前者让你在状态差时硬撑受伤，在状态好时偷懒。RPE让你和身体对话。

## 每次训练操作流程（55-65分钟）

```
Step 0  训练前2小时吃东西（V4新增）
        → 一根香蕉+一杯蛋白粉，或两片全麦吐司+2个蛋
        → 训练前不吃  训练质量打七折

Step 1  热身（8-10分钟）
        → 步骤1-7：关节活动+核心激活（同之前）
        → 步骤8 V4新增：神经激活动作（30秒）
          强度日/爆发日：跳低箱5次或原地高抬腿快交替20秒
          容量日：弹力带侧向走30秒
          如果都没有：原地快速高抬腿20秒

Step 2  力量训练（35-48分钟）
        → 周一：打开Sheet 2「周一 强度日」
        → 周三：打开Sheet 2「周三 容量日」
        → 周五：打开Sheet 2「周五 爆发日」
        → 每个动作后面有RPE列——做完每组根据真实感受判断

Step 3  有氧（10-14分钟，按周调整）
        → 第1周12min  第2周13min  第3周14min  第4周10min
        → 跑步机爬坡：坡度10-12%

Step 4  训练后1小时内吃东西（V4新增）
        → 蛋白粉一勺+香蕉+牛奶
        → 或米饭一碗+鸡胸肉
        → 强度日碳水可以多加一根香蕉

Step 5  冷身（5-8分钟）
        → 6个拉伸动作挨个做
```

## 三个训练日的特点速查

**周一 · 强度日** —— 关键词：重、稳、控
- 主项大口喘气但不力竭（留1-2次余量）
- 辅助用容量方式做（3x10-12）——强度+容量并存
- 训练前神经激活必做
- 训练后碳水吃够50g

**周三 · 容量日** —— 关键词：泵、累、充血
- 短间歇（45-60秒），高次数，肌肉像气球
- 重量偏轻但代谢压力是三天里最大的
- 训练前2h碳水吃够30-40g

**周五 · 爆发日** —— 关键词：快、轻、弹
- 追求启动速度而非重量
- 爆发向心+慢离心——'拉快放慢'
- 训练完不应该很累——如果累了说明重量太重
- 弹力带是爆发日的好朋友

## 减载周——第4周做什么

**核心原则：动但不同**——换动作，不是做轻一点。

| 原来做什么 | 减载周换成什么 |
|-----------|-------------|
| 哑铃卧推 | 俯卧撑（3秒下/爆发推） |
| 器械推肩 | 弹力带推肩（站姿） |
| 高位下拉 | 弹力带高位下拉（单臂） |
| 分腿蹲 | 高脚杯深蹲（轻哑铃） |
| 臀推 | 自重臀桥+弹力带臀桥 |
| RDL | 早安式体前屈（轻量） |

**所有动作统一：2-3组，RPE 4-5。练完应该感觉"我今天练了什么？"——这就是理想减载感受。**

## 季度全休周——每3个月1次

即使每月减载周，3个月连续训练仍会累积**深层系统性疲劳**。

**全休周做什么**：
- 散步30-40分钟/天
- 泡沫轴放松+大量拉伸
- 游泳、瑜伽（如果有条件）
- 不碰器械、不碰哑铃、不做力量训练

**心态**：这不是退步——这是蓄力。学会休息的人才能长期进步。

## 进阶判断规则速查

| 信号 | 做法 |
|------|------|
| 强度日RPE只有7（太轻） | 下次下肢+2.5-5kg，上肢+1-2.5kg |
| 强度日RPE到10（力竭/变形） | 退回上周重量 |
| 容量日'不怎么累' | 缩短休息10秒，或次数+2 |
| 爆发日速度变慢 | 减重1-2kg——速度优先于重量 |
| 减载周感觉太轻松 | 这是对的 |
| 膝盖锐痛 | 当天换无膝压组合 |
| 体重没变力量涨 | 最好信号！继续 |
| 3个月到 | 全休1周 |
