## 核心周期逻辑（先理解这个再看方案）

```
一年的周期框架：

大周期（1年）
  身体重组，从78kg→70-72kg，力量水平稳定提升
  
  中周期（1季度）
    Q1: 月1=容量月  月2=强度月  月3=整合月
    每月4周内部都是：容量→强度→强度→减载
    
    小周期（1个月=4周）
      第1周  容量周  组数+1，次数取上限，重量60-65%
      第2周  强度周  重量递增，次数减少，组数回归正常
      第3周  强度周  继续加重，冲击80-85%极限
      第4周  减载周  重量50-60%，组数减半，主动恢复
```

**为什么要这样设计？**
- 容量周先建立动作储备和身体做功能力——像先把水池挖大
- 强度周再往里加水（重量）——水池大了水才能倒更多
- 减载周让身体消化前三周的刺激——超量恢复发生在休息时，不是训练时

**上下肢为什么加重不一样？**
- 上肢关节精细（肩、肘、腕），一次加太多→关节风险大
- 下肢肌群大、潜力大、关节粗壮，能承受更大增幅
- 这个原则贯穿所有周期——上肢每次+1-2.5kg，下肢每次+2.5-5kg

## 4周操作日历

### 第1周 · 容量周

**目标**：建立动作储备，累积训练容量

**怎么调**：
- 每个动作的组数在基准“+1组”（原来4组变5组，原来3组变4组）
- 次数**取上限**（原来8-10次→做10-12次）
- 重量**降到60-65%*极限重量
- 组间休息**缩短15秒**（增加密度）

**主观感受**：练完应该感觉"量很大，有点累，但不是力竭的那种累"——泵感强，肌肉充血明显。

**记录重点**：把所有动作的重量+组数+次数记下来——这是整个周期的基准线。

### 第2周 · 强度周（上）

**目标**：把容量周的储备转化为力量

**怎么调**：
- 组数**回归正常**（5组→4组，4组→3组）
- 次数**降到中下限**（10-12次→8-10次）
- 重量**增加**——上肢+1-2.5kg，下肢+2.5-5kg
- 组间休息**恢复到正常**（75-90秒）

**主观感受**：第1周的容量积累应该在这一周体现出来——重量比之前重但动作质量不降。

### 第3周 · 强度周（下）

**目标**：冲击本周期力量上限

**怎么调**：
- 继续加重到**80-85%极限**
- 次数进一步降到**6-8次**（主项）
- 已达器械上限的动作→切换到突破策略（慢离心/1.5次法/单侧变式）
- 详见Sheet 4的「第3周调整速查表」

### 第4周 · 减载周

**目标**：主动恢复，让身体超量恢复

**怎么调**：
- 重量**降至50-60%**极限（约第1周的70-80%）
- 每动作**减1-2组**
- 次数**减少30-40%**
- 有氧也一起减：坡度8%，速度4km/h，10分钟

**主观感受**：练完应该感觉"没练够"——**这正是减载周该有的感觉**。如果减载周你还觉得很累，说明前三周强度不够或你没按减载周的要求降。

> ⚠️ **减载周不等于完全不练**——完全不练会让身体脱敏。低强度低容量保持神经适应性，同时让身体全面修复。

## 每次训练的操作流程（55-65分钟）

```
Step 1  热身（8分钟，所有周期通用）
        → 打开Sheet 5「有氧热身冷身」，从步骤1-8挨个做
        → 如果膝盖有不适：第2步靠墙静蹲必做
        → 如果有泡沫轴：先滚股四头肌和髂胫束各30秒

Step 2  力量训练（35-48分钟，取决于哪一周）
        → 容量周：打开Sheet 2，组数+1
        → 强度周：打开Sheet 3/4，重量递增
        → 减载周：全部减量，参考Sheet 4减载周示例

Step 3  有氧（10-15分钟，按周期调整）
        → 容量周：12-15min，坡度10%
        → 强度周：12-15min，坡度10%
        → 减载周：10min，坡度8%，速度4km/h

Step 4  冷身（5-8分钟，所有周期通用）
        → 6个拉伸动作挨个做
```

## 器械上限突破策略（7种）

哑铃只有20kg，当重量不够用时，按周期选择不同策略：

| 策略 | 操作 | 最佳使用周期 |
|------|------|------------|
| 慢离心 | 3-4秒下放+爆发推起 | 所有周期 |
| 1.5次法 | 下放→起一半→再下放→完全起身 | 容量周和强度周 |
| 缩短休息 | 90秒→45秒 | 容量周 |
| 增加次数 | 8-10次→15-20次 | 容量周（核心策略） |
| 单侧变式 | 单腿RDL、单臂划船 | 强度周（首选） |
| 弹力带叠加 | 哑铃+弹力带双阻力 | 强度周突破 |
| 预疲劳法 | 孤立力竭→复合动作 | 容量周进阶 |

## 进阶判断规则——分周期版

| 如果在容量周... | 做法 |
|------|------|
| 所有组轻松完成目标次数上限 | 下周（强度周）可以放心加重 |
| 做到最后1-2组力竭且动作变形 | 下个容量周组数不增加，维持当前容量 |

| 如果在强度周... | 做法 |
|------|------|
| 新重量轻松完成 | 下一强度周（或下周期）继续加 |
| 新重量完成但最后2组非常吃力 | 维持，不加重——这就是最佳区间 |
| 无法完成目标次数下限或动作变形 | 退回上一周重量 |

| 如果在减载周... | 做法 |
|------|------|
| 觉得"太轻松了，想冲" | **忍住。**轻松是减载周的目的，不是问题 |
| 减载周结束精力充沛 | 进入下周期容量周，重新出发 |

## 常见问题

**Q：容量周练完感觉很累，强度周反而没那么累？**
A：正常。容量周追求的是"量的累积"——泵感和代谢压力；强度周追求的是"质的突破"——神经系统的挑战。两者的疲劳类型不同。

**Q：减载周真的有必要吗？我感觉浪费了一周。**
A：减载周不是浪费——是投资。前3周的刺激在减载周里"消化"，超量恢复在减载周结束后发生。跳过减载周=持续累积疲劳=3-4个周期后必撞墙。37岁的身体尤其需要这个周期性的修复窗口。

**Q：容量周和强度周如果我搞混了怎么办？**
A：打开Excel看当前周的Sheet就行。每周训练前花5分钟看一遍该周的计划——已经全写好了。

**Q：减载周结束可以直接进入下个周期的容量周吗？**
A：对。减载周结束后，你应该精力充沛地进入下个周期的容量周。如果减载周结束仍然疲劳→延长减载1周（再做1周轻量训练），再进入新周期。

**Q：下个周期的容量周比这个周期加重吗？**
A：看情况。一般下个周期的容量周重量 = 这个周期强度周重量的70-75%。你已经在更高的基准线上开始新的容量积累了——这就是周期训练的长期效果。
