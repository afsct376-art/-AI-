## 四周节奏

| 周 | 阶段 | 做什么 |
|----|------|--------|
| 第1周 | 唤醒 | 每个动作3组，中等重量。重建动作记忆。 |
| 第2周 | 容量 | 每个动作+1组（3→4组），次数取上限。身体开始适应。 |
| 第3周 | 强度 | 比前两周重1-2片。找到"有点挑战"的感觉。 |
| 第4周 | 减载 | 换动作，轻重量。让身体恢复。拍对比照，量体重腰围。 |

## 每周三次训练

```
周一 = 全身A（推力+下肢主导）
      坐姿推胸 → 推肩 → 腿举 → 髋外展 → 高位下拉 → 举腿

周三 = 全身B（拉力+下肢后侧主导）
      坐姿划船 → 高位下拉(宽握) → 腿弯举 → 臀桥 → 髋内收 → 平板支撑

周五 = 全身C（综合+手臂+额外有氧）
      推胸 → 高位下拉(窄握) → 腿举 → 侧平举 → 绳索下压 → 跑步机多走5-10分钟
```

每个训练日有6个动作。打开Sheet「训练计划」找到今天的格子，按照动作表挨个做。

## 每次训练流程（约60分钟）

```
热身（5分钟）
  → 踏步+摆臂 → 肩绕圈 → 摆腿 → 扶椅深蹲 → 第一个动作轻重量试做

力量训练（35-48分钟）
  → 打开「训练计划」→ 找到今天（全身A/B/C）→ 按表格挨个做

有氧（10-20分钟）
  → 跑步机爬坡走：坡度8-10%，速度4.5-5km/h
  → 全身C日多走5-10分钟

拉伸（5分钟）
  → 6个动作挨个做——别跳过
```

## 饮食——把上次的成功捡回来

你上次40天瘦7.5kg，饮食是主力。下面这些你大概率不陌生：

### 每餐怎么吃

| 餐次 | 吃什么 |
|------|--------|
| 早餐 | 2个蛋+牛奶+全麦吐司/燕麦 |
| 午餐 | 米饭一拳+手掌大肉/鱼+蔬菜半盘 |
| 晚餐 | **粗粮一拳**（糙米/红薯/玉米/南瓜）+肉/豆腐+大量蔬菜 |
| 训练后 | 蛋白粉+水或牛奶（30分钟内） |

**核心**：三餐每餐有蛋白质。晚餐碳水换成粗粮。戒含糖饮料。先吃菜再吃肉最后吃饭。

### 如果吃外卖

✅ 沙县鸡腿饭(去皮)+烫青菜 | 麻辣烫清汤多菜少丸子 | 便利店鸡胸肉+蛋+豆浆

❌ 炒饭/炒面/盖浇饭 | 油炸类 | 含糖饮料

## 经期前后

- **经前一周**：体重涨1-3kg=正常水肿，不是胖了。想吃甜的=激素波动，吃黑巧克力/水果。训练降强度但尽量去。
- **经期1-2天**：休息或只走路。第3天后恢复轻量。

---

## 掉秤预期

- 第一周可能掉1-2kg（水分为主）
- 之后稳定在0.5-1kg/周
- 某周没变：先看是不是经前。再检查饮料和晚餐碳水
- 你上次40天平均每周约1.3kg——这次不用那么激进

## 遇到情况

| 情况 | 怎么办 |
|------|--------|
| 太累了不想练 | 走5000步+拉伸——做了就算。别连续跳2次 |
| 吃多了 | 明天继续。一顿不会胖 |
| 训练后48h还酸痛 | 正常——重新练前两周会酸。关节疼告诉我 |
| 膝盖/腰/关节疼 | 立即停那个动作。疼超2天告诉我 |
| 状态极差 | 打开「备用与复盘」看降级方案 |

## 拍照——今天就拍

穿运动内衣+紧身裤。三张：正面（手机与肚脐同高）、侧面（手机与腰同高）、背面（手机与肩胛骨同高）。不要吸肚子不要找角度。4周后同一位置同一光线再拍。
