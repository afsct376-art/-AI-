## 专业术语速查表

方案里用了几个专业术语——你大概率都知道，但万一有模糊的翻一下。

| 术语 | 全称 | 白话解释 |
|------|------|---------|
| **DUP** | Daily Undulating Periodization（每日波动周期） | 每天练的东西不一样。周一重、周二多、周四快、周五细——不让身体适应同一种节奏，进步更持续。 |
| **AMRAP** | As Many Reps As Possible（尽可能多做一次） | 最后一组不限定次数，做到动作快变形了就停。用来测试你今天到底能推多少个——比"规定做5次"更真实。 |
| **RPE** | Rating of Perceived Exertion（自感用力程度） | 做完一组后凭感觉打分。1分=躺着不动，10分=拼了老命。你训练时做到8-9分就行——做完还剩1-2次力竭的感觉。 |
| **超级组** | Superset | 两个动作连着做，中间不休息。比如做完深蹲马上做腿举，做完一组才休息。用来压缩时间、增加代谢压力。 |
| **预疲劳法** | Pre-exhaustion | 先做孤立动作让目标肌肉"预热"充血，再做复合动作深度刺激。比如先做侧平举让肩膀充血，再做卧推——肩膀会更有感觉。 |
| **减载周** | Deload Week | 每4周安排1周轻量训练，让身体消化前3周的刺激。不是偷懒——是投资。超量恢复发生在休息时，不是训练时。 |
| **皮质醇** | Cortisol | 压力激素。训练时间太长、压力太大时身体会分泌它——它会分解肌肉。所以腿日控制在45分钟内，就是为了不让皮质醇起来。 |

## V2 相比 V1 升级了什么

| 优化点 | V1 | V2 |
|--------|----|----|
| 推日卧推 | 5x5 | **4x5+1AMRAP**——前4组标准刺激 最后一组探底找真实极限 |
| 拉日结构 | 不分宽/厚 | **宽度优先版和厚度优先版每周轮换**——背=视觉宽度+立体感 |
| 腿日时长 | 无时间限制 | **45分钟限时**——代谢压力黄金窗口 超时=皮质醇分解肌肉 |
| 上肢日 | 补弱点 | **预疲劳法**——孤立充血→复合深度刺激 突破瓶颈 |
| 减载周 | 做轻一点 | **换动作**——同样动作做轻量=关节应力路径相同 不是真休息 |
| 营养 | 方向性 | **训练前后窗口精确化+教课日营养** |
| 长期规划 | 无 | **季度全休周**——每3个月1周完全休息 职业保护 |

## 你是教练——这份方案是写给你的

你每天在健身房教别人。杠铃卧推、深蹲、划船——这些不需要我教你。但你可能很久没有为自己系统做过计划了。

这份方案按"同行标准"写：
- 没有基础动作教学——你都知道
- 有DUP波动周期逻辑——每天不同刺激
- 有RPE进阶体系——不只看重量 看当天感觉
- 有动作分类说明——让你理解每个动作在整体框架中的角色
- 有渐进决策树——让你像做实验一样训练 每次都有数据可复盘
- 有自主调节框架——用了这套 以后你可以自己迭代

**这套框架你可以用在任何会员身上。**

## 年度周期框架——我现在在哪 接下来去哪

没有年度视角的周期设计 = 只有战术没有战略。你需要知道自己在哪条线上。

| 季度 | 阶段 | 目标 | 训练特点 |
|------|------|------|---------|
| **Q1（1-3月）** | 积累期 | 建立动作基准 提升容量 | DUP 4天 3周渐进+1周减载。第3个月末：1周全休 |
| **Q2（4-6月）** | 强化期 | 提升力量 增加肌肉量 | 重量递增 RPE 8-9。第6个月末：1周全休 |
| **Q3（7-9月）** | 峰值期 | 冲击极限 测试1RM | 强度优先 RPE 9-10（偶尔）。第9个月末：1周全休 |
| **Q4（10-12月）** | 维持/过渡 | 维持力量 降低训练压力 | 容量为主 RPE 6-7。年底：2周全休（彻底重置） |

**你现在在哪？** 如果刚开始用这套方案——你在Q1积累期。先跑完一个4周周期，把基准数据建起来。

## 一周四天——DUP波动 V2版

| 天 | 主题 | V2核心变化 | RPE | 核心感觉 |
|----|------|----------|-----|---------|
| **周一** | 推日（大重量） | **4x5+1AMRAP** | 8-9 | 前4组标准 第5组探底 |
| **周二** | 拉日（宽/厚轮换） | **第1/3周=宽度 第2周=厚度** | 7-8 | 背的宽度+厚度双维度 |
| **周四** | 腿日（代谢 45min限时） | **深蹲+腿举/弯举超级组** | 7-8 | 短间歇 45min内完成 |
| **周五** | 上肢日（预疲劳） | **孤立充血→复合深度刺激** | 7-8 | 先充血再深度刺激 |

### 为什么每天不同主题？

DUP波动——神经系统每周面对4种不同刺激，适应性更慢，进步更持续。普通计划8周后身体就适应了。DUP让你在同一个4周内体验4种不同节奏——你教会员时也应该用这个逻辑。

## 每个训练日的动作分类逻辑

你是教练——你不需要"动作教学"，但你需要**理解每个动作在整体框架中的角色**。这样以后你可以自己替换动作，而不破坏训练逻辑。

### 周一 · 推日 —— 动作分类

| 角色 | 动作 | 为什么选它 |
|------|------|-----------|
| **主项（复合 神经驱动最高）** | 杠铃卧推 4x5+1AMRAP | 水平推类动作中 卧推能调动最多运动单位 是上半身力量的"标尺" |
| **辅助1（垂直推 补肩部）** | 哑铃推肩 4x8-10 | 卧推主要练胸和肱三头 肩前束刺激不够。推肩补上这个缺口 |
| **辅助2（上胸变式）** | 上斜哑铃卧推 4x8-10 | 和平板卧推不同角度——上胸+肩前束 补充主项的盲区 |
| **辅助3（三头孤立）** | 双杠臂屈伸 3x8-12 | 卧推后三头已疲劳——这时候做臂屈伸 三头会第一个力竭 |
| **收尾（代谢补充）** | 绳索飞鸟+三头下压 | 在主项和辅助之后 继续刺激胸肌和三头的代谢——泵感收尾 |

### 周二 · 拉日（宽度优先版 第1/3周）—— 动作分类

| 角色 | 动作 | 为什么选它 |
|------|------|-----------|
| **主项（垂直拉 宽度）** | 宽握高位下拉 4x10-12 | 宽握=背阔肌外沿——宽度=视觉冲击力。这是宽度日 不是重量日。 |
| **辅助1（拉伸角度）** | 直臂下压 3x15 | 背阔肌的全程拉伸——从头顶下压至大腿。和下拉互补。 |
| **辅助2（水平拉 厚度补充）** | 窄握坐姿划船 3x12-15 | 窄握=中背部厚度。宽度日也需要厚度——但以轻量补充为主。 |
| **下肢（后链）** | 罗马尼亚硬拉 4x8-10 | 拉日的固定动作——腘绳肌+臀。和腿日深蹲互补。 |
| **收尾（肩袖+二头）** | 面拉+弯举 | 肩后束健康+二头泵感——拉日最后给手臂充血。 |

### 周二 · 拉日（厚度优先版 第2周）—— 动作分类

| 角色 | 动作 | 为什么选它 |
|------|------|-----------|
| **主项（水平拉 厚度）** | 杠铃俯身划船 4x8-10 | 厚度=立体感。重量可以比宽度日重——杠铃划船能承载更大负荷。 |
| **辅助1（不同角度划船）** | T杠/单臂哑铃划船 3x10-12 | T杠=中背部厚度。单臂=左右独立 纠正不平衡。 |
| **辅助2（宽度维持）** | 宽握高位下拉(轻量) 3x15 | 厚度日也保持宽度刺激——轻重量 不追求力竭。 |
| **下肢（后链）** | 罗马尼亚硬拉 4x8-10 | 同宽度日——RDL是拉日固定动作。 |
| **收尾（肩袖+二头）** | 面拉+弯举 | 同宽度日。 |

### 周四 · 腿日（45min限时）—— 动作分类

| 角色 | 动作 | 为什么选它 |
|------|------|-----------|
| **主项（复合 下肢核心）** | 杠铃深蹲 4x8-10 60s休息 | 深蹲是下肢力量的标尺。短间歇=代谢压力——和力量举的长间歇不同。 |
| **超级组（代谢密度翻倍）** | 腿举+腿弯举 各3x15-20 不休息 | A做完直接做B。股四头+腘绳肌同时充血——代谢压力拉满。 |
| **收尾1（小腿）** | 站姿提踵 3x15-20 | 小腿=高频次短间歇——在腿日疲劳状态下完成。 |
| **收尾2（核心）** | 悬垂举腿 3x12-15 | 腿日核心——在代谢疲劳状态下稳定核心。 |

### 周五 · 上肢日（预疲劳法）—— 动作分类

| 角色 | 动作 | 为什么选它 |
|------|------|-----------|
| **预疲劳超级组（先做！）** | 侧平举+三头下压 各3x15 交替不休息 | 先让侧肩和三头充血——突破"做完了但目标肌群还没感觉"的瓶颈。 |
| **主项（三头主导复合）** | 窄距卧推/地板卧推 4x8-10 | 预疲劳后三头已经充血——这时候窄距卧推 三头会第一个力竭。重量不是指标 泵感才是。 |
| **辅助（垂直拉 维持背）** | 高位下拉(宽握) 4x10-12 | 上肢日保持背阔肌刺激——和拉日同动作。 |
| **收尾（肩袖+功能）** | 面拉+农夫行走 | 面拉保护肩关节健康。农夫行走=握力+斜方+核心——功能性收尾。 |

## 渐进决策树——每次训练前看

没有决策标准的渐进 = 凭感觉训练。有了决策树，你可以**像做实验一样训练**——每次训练都是一个数据点，4周后复盘时能看到清晰的进步轨迹。

### 推日（周一 强度日）

| 情况 | 做法 |
|------|------|
| 前4组5次都完成 且RPE ≤ 8 | ✅ 下次加重2.5-5kg |
| 前4组5次都完成 但RPE = 9 | ➡️ 维持当前重量 |
| 前4组中有1组没完成5次 | ⬇️ 退回上周重量 |
| 连续2周卡在同一重量 | 🔄 换变式：地板卧推/暂停卧推/窄距卧推 |
| AMRAP次数比上周+2次以上 | ✅ 真的变强了——下次加重 |
| AMRAP次数没变或下降 | ➡️ 维持——今天的你不如上周的你 正常 不退步 |

### 拉日（周二 容量日）

| 情况 | 做法 |
|------|------|
| 所有组完成目标次数 且RPE ≤ 7 | ✅ 下次+1组或缩短休息15秒 |
| 所有组完成 但RPE = 8 | ➡️ 维持 |
| 有1-2组没完成目标次数 | ⬇️ 减重或增加休息时间 |
| 宽度vs厚度——哪个进步更快？ | 进步快的保持 慢的下周期多轮一次 |

### 腿日（周四 代谢日）

| 情况 | 做法 |
|------|------|
| 45分钟内完成全部 | ✅ 下周缩短休息5-10秒 |
| 45分钟没完成 | ⬇️ 减少超级组轮数——宁可做少 不要超时 |
| 深蹲RPE ≤ 7 轻松完成 | ✅ 下周+2.5-5kg |
| 超级组做到第2轮就力竭 | ➡️ 维持——代谢压力的目的达到了 |

### 上肢日（周五 弱点日）

| 情况 | 做法 |
|------|------|
| 预疲劳后窄距卧推重量比预期低 | 正常——预疲劳的目的就是这个。重量不是指标 泵感才是 |
| 预疲劳超级组做完 侧肩和三头没有泵感 | ⬆️ 加重或缩短休息——不够充血 |
| 窄距卧推连续2周没进步 | 🔄 换变式：JM Press/仰卧臂屈伸 |

### 通用规则

| 信号 | 做法 |
|------|------|
| 连续2周睡不好+训练状态差 | 提前进入减载周——你是教练 知道过度训练的代价 |
| 连续4周某个动作没进步 | 换变式——同肌群不同角度重新刺激 |
| 减载周感觉"太轻松" | 这是对的——减载周不轻松=前三周没练到位 |

## 主动恢复——休息日不是"躺着恢复"

你每天在健身房教课，身体一直在"低强度消耗"状态。如果不做主动恢复，你可能会陷入"练了但没恢复"的循环。**主动恢复不是"可选项"——是"必修课"。**

### 休息日（周三/六/日）—— 主动恢复清单

选1-2项做，15-20分钟：

**1. 泡沫轴全身放松（10分钟）**
```
顺序：小腿 → 股四头肌 → 腘绳肌 → 臀 → 背 → 胸
每个部位滚30秒 找到酸痛点停留15秒深呼吸
```

**2. 髋关节灵活性训练（10分钟）**
```
世界最伟大拉伸 每侧5次
90-90髋拉伸 每侧30秒
青蛙趴 30秒
```

**3. 肩关节保养（10分钟）**
```
YTWL 每个字母10次
面拉 3x15（轻重量）
弹力带肩外旋 每侧15次
```

**4. 低强度有氧（20分钟）**
```
散步 / 游泳 / 休闲骑行
心率控制在120-130bpm——不是训练 是恢复
```

**完全不练也比"轻重量练一下"好——休息日的目标是恢复 不是"顺便练练"。**

## 六个V2升级的底层逻辑

### 1. 推日：4x5+1AMRAP

**5x5的问题**：第五组往往因疲劳累积而动作变形。对于健美选手 动作质量>组数数量。

**V2的做法**：前4组标准5次 RPE 8-9。第5组AMRAP——做到技术变形前停。记录次数。

**AMRAP是每周进步的标尺**：第1周做了几次？第3周同样重量做了几次？数字涨了=你真的变强了。

```
示例：
第1周 80kg 4x5 + AMRAP=7次
第3周 85kg 4x5 + AMRAP=6次
→ 重量涨了 AMRAP仍6次 你在变强
```

### 2. 拉日：宽度/厚度每周轮换

健美选手的背需要**宽度（视觉冲击力）和厚度（立体感）**同时发展。

- **宽度优先**（第1/3周）：宽握高位下拉主项 感受背阔肌向外展。
- **厚度优先**（第2周）：杠铃划船主项 重量可以比宽度日重。

每周轮换=两个维度都得到足够刺激。

### 3. 腿日：45分钟限时

代谢压力是增肌的有效手段。但**不加控制的代谢压力=皮质醇飙升**。

45分钟是代谢压力增肌的黄金窗口——刺激够了 皮质醇还没起来。超时=皮质醇开始分解肌肉 得不偿失。

如果45分钟没完成：减少超级组轮数——宁可做少 不要超时。

### 4. 上肢日：预疲劳法

先做侧平举+三头下压超级组让目标肌群充血→再做窄距卧推深度刺激。突破"复合动作做完了但目标肌群还没感觉"的瓶颈。

**预疲劳后重量比预期轻=正常。重量不是指标 泵感才是。**

### 5. 减载周：换动作

同样动作做轻量=关节和神经仍承受相同应力路径。不是真正的休息。

V2减载周换动作：杠铃卧推→俯卧撑 杠铃划船→弹力带划船 杠铃深蹲→高脚杯深蹲。换动作=给关节不同的应力分布 同时保持血液循环和神经激活。

### 6. 季度全休

即使有减载周 连续3个月DUP训练仍累积系统性疲劳。你教一天课=恢复压力比普通人大一倍。

每3个月1周完全休息（散步/游泳/拉伸/泡沫轴 不碰器械）。全休周不是偷懒——是职业保护。

## 状态差时的心理策略

你最大的挑战不是"不知道怎么练"——是"教了一天课，已经没有精力给自己练了"。

| 情况 | 做法 |
|------|------|
| **"今天不想练"** | 去健身房 只做热身+主项3组。如果做完3组还是不想练→回家 今天算完成。如果做完3组感觉来了→继续。你只是需要"开始" |
| **"连续几天状态差"** | 检查睡眠和饮食。睡眠不足→优先补觉 训练降级。饮食没吃够→加碳水 尤其是训练前后 |
| **"连续2周没进步"** | 不是你的问题 是计划需要调整。可能原因：累积疲劳/动作需要换变式/热量没吃够。解决方案：减载1周+换动作变式+加碳水 |
| **"教课太累 不想练自己的"** | 把训练当成"给自己充电"。你每天在输出（教课） 训练是唯一属于你的输入时间。哪怕只练20分钟 也比不练好 |
| **"觉得自己练得没会员好"** | 你是教练 不是选手。你的训练不需要和别人比——只需要比上周的自己好。AMRAP+1次 深蹲+2.5kg 都是进步 |

## 营养——训练前后窗口精确化 V2

你每天在健身房教课 身体一直处于低强度消耗状态。如果不注意训练前后营养窗口 可能会陷入"练了但没恢复"的循环。

| 时机 | 吃什么 | 为什么 |
|------|--------|--------|
| **训练前2小时** | 碳水40-50g+蛋白20-25g（香蕉+蛋白粉+全麦吐司2片） | 提供训练能量 防止训练中分解肌肉。你教课消耗大 练前不吃=练到一半没力气 |
| **训练中** | 水+电解质 500-750ml | 维持表现 防抽筋——尤其腿日和推日 |
| **训练后1小时内** | 碳水50-60g+蛋白30-35g（米饭一碗+鸡胸肉200g/蛋白粉2勺+香蕉+牛奶） | 补充糖原 启动肌肉修复。训练后1h=蛋白质吸收效率最高时段 |
| **教课日（非训练日）** | 每4-5小时吃一顿含蛋白质的餐 | 你教课也在消耗——不要等到训练日才吃够。保持蛋白质摄入稳定 |
| **休息日** | 碳水比训练日少20-30% | 不练不需要那么多能量——自然少一点。蛋白质不变 |

**全天方向**：总热量2700-2900 蛋白质160-180g(2.0-2.2g/kg) 碳水320-360g 脂肪60-70g 水3-4L。干净增肌——不做脏增肌。你教一天课 碳水是真正的能量来源 不要害怕吃碳水。

## 自主调节框架——这套你可以带走

```
1. 定分化方式：每周能练几天→推拉腿/上下肢/全身。你4天=推+拉(宽/厚)+腿(代谢)+上肢(预疲劳)

4. 定RPE范围：百分比是理论值 RPE是当天真实状态
5. 每月复盘：AMRAP次数+力量+体重+照片
```

这套逻辑——你在西宁健身房可以用在任何一个会员身上。
