## 第一周 · 照着清单打勾

封面页的「第一周每日清单」有 10 项——每天早上看一眼，做到一项打一个勾。

| # | 今天要做 |
|---|---------|
| 1 | 早上起床喝一杯水（500ml） |
| 2 | 早餐：2个蛋 + 牛奶 + 一片全麦吐司 / 一小碗燕麦 |
| 3 | 水壶放旁边，上午一壶下午一壶，喝够3L |
| 4 | 午餐：米饭一拳 + 一份肉/鱼 + 蔬菜多夹 |
| 5 | 不喝含糖饮料——奶茶换无糖茶，可乐换气泡水+柠檬 |
| 6 | 步数攒到8000步（看"8000步拆解"） |
| 7 | 晚餐：粗粮一拳 + 一份肉/豆腐 + 大量蔬菜 |
| 8 | 晚饭后散步20分钟——最重要的一段步数 |
| 9 | 睡前不吃了，饿了喝温水或吃黄瓜/小番茄 |
| 10 | 睡够7小时——定好闹钟，手机放远一点 |

**做到任意6项——今天就是成功的一天。** 做不到6项也没关系，明天继续。不需要完美，只需要持续。

## 8000步——拆开看一点都不大

（这个在封面页能看到完整版）

| 场景 | 步数 | 怎么做 |
|------|------|--------|
| 早上通勤多走一站 | +1500步 | 提前一站下车走过去 |
| 午休溜达10分钟 | +2000步 | 吃完饭别马上坐 |
| 下午接水/上厕所多走 | +1000步 | 绕远路攒起来 |
| 晚饭后散步20分钟 | +2500步 | 最重要的一段 |
| 晚上在家走动/收拾 | +1000步 | 做家务也算 |
| **合计** | **≈8000步** | 走不够 5000 也比不走强 |

不用一口气走完。碎片时间凑够就行。手机计步自动帮你加。

## 如果第一周觉得同时做四件事太难

封面页有一个「每周只加一件事」的备用路径：

| 周 | 只加一件事 | 说明 |
|----|----------|------|
| 第1周 | 只戒含糖饮料 | 其他三件能做就做，做不到不批评自己 |
| 第2周 | 戒饮料 + 8000步 | 第一个习惯稳了再加 |
| 第3周 | 戒饮料 + 8000步 + 3L水 | 前两个已经不用"坚持"了 |
| 第4周 | 四件事全做到 | 你已经有运动习惯的人了 |

**慢就是快。** 一个月养成一个习惯，比一个月逼自己四件事然后放弃——效果好十倍。

## 如果第一周完全没做到——一件都没做到

**没关系。** 这不是"失败"——是"还没开始"。明天重新来。不用等周一，不用等下个月。从"今天喝够3L水"开始，就这一件事。做到了，第一周就算你过。

## 关于掉秤速度——提前知道就不焦虑

- **第一周**可能会掉 1-2kg（主要是水分——戒掉含糖饮料后身体开始排水）
- **第二周开始**变慢，0.5-1kg/周——这才是真正的脂肪在掉
- **如果某周体重没变**：先看是不是经前（水肿1-3kg正常），再看是不是吃了咸的，最后看腰围——腰围小了就是进步
- 你用了好几年胖到 97kg，不可能几周瘦到 70kg。慢就是快，稳才能不反弹

## 第二周 · 走进健身房

训练前打开 Sheet **「第2-4周训练」**，先看顶部的 **"怎么选重量"**：

1. 先挂最轻的片，做一组试试
2. 如果 15 次做完"完全没感觉"→加一片
3. 如果做到第 12 次就做不动了→减一片
4. **理想状态**：做完 15 次，最后 3 次有点累，但动作不变形

第2周三个训练日都做同样的 6 个动作（全部固定器械，轨道固定，一个人安全）。目标不是"练到累"——是学会每个器械怎么用、找到发力感。

## 拍照——今天就拍

打开 Sheet **「热身有氧拉伸」** 末尾的拍照标准，今天拍三张：

| 拍什么 | 怎么拍 | 要点 |
|--------|--------|------|
| 正面照 | 自然站立，双手放两侧 | 手机与肚脐同高，全身入镜 |
| 侧面照 | 侧身站立，手臂自然下垂 | 手机与腰同高 |
| 背面照 | 背对镜子，双手叉腰 | 手机与肩胛骨同高 |

穿运动内衣+紧身裤。**不要吸肚子！不要找角度！** 真实的照片才有对比价值。4 周后同一位置同一光线再拍一组。

## 饮食——不用称克数

### 每餐怎么吃

| 餐次 | 吃什么 | 要点 |
|------|--------|------|
| 早餐 | 2个蛋 + 牛奶 + 全麦吐司/燕麦 | 蛋白质吃够上午不饿。蛋黄别丢 |
| 午餐 | 米饭一拳 + 手掌大肉/鱼 + 蔬菜半盘 | 清蒸/炖/卤，避开油炸。先菜后肉后饭 |
| 晚餐 | **粗粮一拳** + 手掌大肉/豆腐 + 大量蔬菜 | 粗粮=糙米/红薯/玉米/南瓜 |
| 训练后 | 蛋白粉+水或牛奶 | 30分钟内喝 |

**如果今天没有粗粮**：白米饭一拳也可以——吃对总量比吃对种类重要。

**睡前饿了**：喝温水/吃黄瓜/小番茄。不要硬扛——硬扛的结果是半夜翻冰箱。

### 外食/外卖怎么选

**✅ 选这些**：沙县鸡腿饭(去皮)+烫青菜 | 麻辣烫清汤多菜少丸子 | 轻食沙拉酱汁另放蘸不浇 | 便利店鸡胸肉+蛋+豆浆+玉米

**❌ 避开这些**：炒饭/炒面/盖浇饭 | 油炸类 | 含糖饮料 | 勾芡/红烧/糖醋类

不要求顿顿完美。一天里有一顿吃对了——就比昨天好。

## 经期前后——提前知道就不慌

**经前一周（PMS）**：
- 体重涨 1-3kg→正常水肿，不是胖了。经期结束水就排掉
- 食欲暴增想吃甜→正常激素波动，可以吃，选黑巧克力(70%+)/水果/无糖酸奶
- 不想训练→降强度，但尽量保持去健身房——哪怕只做热身+有氧。经前一周"维持"就是胜利

**经期第 1-2 天**：休息或只走路。第 3 天后 OK 就恢复轻量。

## 遇到这些情况

| 情况 | 怎么办 |
|------|--------|
| 今天太累了不想练 | 走 5000 步+拉伸——做了就算。但别连续跳 2 次 |
| 这周吃多了/喝了奶茶 | 明天继续，别等周一。一顿不会胖——连续一周才会 |
| 训练后 48 小时还酸痛 | 零基础前两周就会酸，正常。关节疼不是肌肉疼→告诉我 |
| 体重一周没掉 | 先看是不是经期前后。不是的话检查饮料和步数 |
| 膝盖/腰/关节疼 | 立即停那个动作，换不疼的。疼超 2 天告诉我 |
| 状态极差/加班/事情多 | 打开「备用与复盘」看降级方案 |

## 4 周后怎么知道自己进步了

**不要只看体重秤。** 以下全是进步：

| 看什么 | 可能的变化 |
|--------|----------|
| 体重 | 掉了 1-4kg |
| 腰围 | 小了 2-5cm（比体重更真实） |
| 照片 | 翻出第一天的三张——照片不会骗人 |
| 衣服 | 裤腰松了，上衣不绷了 |
| 体力 | 从 8000 步气喘到 10000 步不累 |
| 自信 | 从"不敢进健身房"到"知道每个器械怎么用" |
| 习惯 | 走路/喝水/不喝甜饮料——已经不用"坚持"了 |

**最后一条才是永远不会反弹的保证。**
