## 打开Excel 第一眼——术语速查就在封面上

封面第一个Sheet叫「从这里开始」——第一屏就是七个专业术语的速查表。你练了3-4年 不会问我"AMRAP是什么""RPE怎么打"——但万一有模糊的 翻一下。

**V4在术语表里加了两句话：**

> **高翻50%** = 你高翻1RM的50%。如果没测过1RM：用你能流畅完成5次、不砸锁骨、不弯肘的最大重量 × 0.5。

> **速度变慢** = 第3次推起时明显比第1次慢 或者需要"硬挤"才能推上去。出现=太重了 减重5%。

## 快速开始——打开 Sheet「示例检查FAQ」

### 第一周不知道从哪开始？

**直接照着示例做。** Sheet「示例检查FAQ」里有三个训练日各一个完整示例——步骤、动作、组数次数、参考重量（重量按你自己的实际调整）。

如果不确定85%1RM是多少：用你能做5次、但第5次非常吃力的重量。那个重量大概就是你的85%1RM。

### 每周花1分钟填检查清单

练完周五后花1分钟填6项：体重趋势、AMRAP次数、深蹲重量、高翻技术、睡眠、本周感受。4周累计 回头能清晰看到趋势。

## 高翻——这次是重头戏 每次训练前都做

高翻是技能 不是力量训练。技能学习需要高频低强度——不是每周一次高强度。

| 训练日 | 高翻 | 加重规则 |
|--------|------|---------|
| 周一 | 5x3@50% 轻量练速度 | 保持轻量——不加重 |
| 周三 | 5x3@50-55% 稍重 | **只在周三加重 且技术流畅时才+2.5kg** |
| 周五 | 5x3@50% 轻量练速度 | 保持轻量——不加重 |

第1周全50%。第2周周三55%。第3周周三60%。第4周全50%。

### 高翻判断规则

| 信号 | 做法 |
|------|------|
| 动作流畅 杠铃轨迹直 接杠稳 | 周三可以+2.5kg |
| 手臂过早发力弯肘 | 减重 练髋爆发+耸肩 |
| 接杠砸锁骨 | 不加重 练前蹲+耸肩 |
| 连续2周流畅 | 周三+2.5kg |
| 今天状态差 | 不练高翻——改耸肩+前蹲 |

## 速度日——V4核心创新

8x3@60-65%爆发力——练神经驱动 不是练肌肉。每组3次 爆发推起 控制下放。不累积疲劳——周五练完周末能恢复。

**速度变慢=第3次明显比第1次慢 或需要硬挤。出现这种情况=减重5%。**

|------|------|----------|
| **第1-4周（本周期）** | 高杠深蹲 | 和接杠姿势相似 |
| 第5-8周 | 低杠深蹲 | 强化后侧链 启动更有力 |
| 第9-12周 | 前蹲 | **最值得练——最接近高翻接杠** |
| 第13-16周 | 箱式深蹲 | 练底部爆发力 突破粘滞点 |

## 第4周减载——换动作 不是做轻一点

| 原动作 | 第4周换成 |
|--------|----------|
| 高翻(重) | 杠铃耸肩+前蹲（练分解） |
| 杠铃卧推 | 俯卧撑（3秒下+爆发推） |
| 杠铃深蹲 | 高脚杯深蹲 |
| 杠铃划船 | 哑铃单臂划船 |
| 负重引体 | 高位下拉（轻量） |

所有动作2-3组 轻重量。做完应该感觉"没练够"——这就对了。

## 遇到情况

| 信号 | 做法 |
|------|--------|
| AMRAP比上周+2次 | 变强了 下次+2.5-5kg |
| AMRAP没变或下降 | 维持重量 不退步 |
| 速度卧推速度变慢 | 减重5%——速度>重量 |
| 高翻连续2周流畅 | 周三+2.5kg |
| 午间训练没力气 | 查11:30那顿——是不是没吃够 |
| 中午练完下午犯困 | 练后碳水减半碗 蛋白不变 |
| 状态极差 | 热身+高翻+主项3组轻量+拉伸 |

## 自主调节框架——3年基础 你可以自己迭代

1. **定分化**：每周3天→DUP（上体力+下肢力+高翻+速度+容量）
2. **高翻每练必做**：5-8分钟 5x3@50-60%。技能训练 高频低强度
3. **定周期**：3周渐进+1周减载（换动作）4周一循环。深蹲每4周轮换变式
4. **定RPE**：周一8-9（重）周三7-8（重+技能）周五不评RPE（速度日看速度不看RPE）
5. **力型兼备热量决策**：体重趋势+力量趋势+体脂趋势 三个指标联动调整
