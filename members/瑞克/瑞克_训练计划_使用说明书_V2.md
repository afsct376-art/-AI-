## 高翻——这次是重头戏 每次训练前都做

**高翻是技能 不是力量训练。技能学习需要高频低强度——不是每周一次高强度。**

每次训练热身之后 主项之前 练5-8分钟：

| 训练日 | 高翻 | 什么时候加重 |
|--------|------|------------|
| 周一（上体力） | 5x3@50% 轻量练速度 | 保持轻量——不加重 |
| 周三（下肢力） | 5x3@50-55% 稍重 | **周三可以+2.5kg 但只在技术流畅时** |
| 周五（速度+容量） | 5x3@50% 轻量练速度 | 保持轻量——不加重 |

**第1周=50%全轻量。第2周=周三55% 其它50%。第3周=周三60% 其它50-55%。第4周=全部回到50%轻量。**

### 高翻判断规则

| 信号 | 做法 |
|------|------|
| 动作流畅 杠铃轨迹直 接杠稳 | ✅ 周三可以+2.5kg |
| 手臂过早发力弯肘 | ❌ 减重 练三倍伸展 |
| 接杠砸锁骨 | ❌ 不加重 练前蹲+耸肩 |
| 连续2周流畅 | 周三+2.5kg |
| 今天状态差 | 不练高翻——改耸肩+前蹲 |

## 速度日——V2核心创新

**速度卧推8x3@60-65%——练神经驱动 不是练肌肉。每组3次 爆发力推起 控制下放。不累积疲劳——周五练完周末恢复。**

- 速度变慢 = 太重了 减重5%
- 组间只有60秒——不拖堂
- 8组总共24次——容量不低 但不是力竭
- 直接帮助周一的3x5+1AMRAP——速度日=力量日的助跑

## 深蹲变式轮换

| 周期 | 变式 | 特点 | 对高翻帮助 |
|------|------|------|----------|
| **第1-4周** | 高杠深蹲 | 躯干竖直 股四头主导 | 和接杠姿势相似 |
| 第5-8周 | 低杠深蹲 | 后侧链主导 | 强化启动 |
| 第9-12周 | 前蹲 | 核心要求最高 | **最值得练——最接近高翻接杠** |
| 第13-16周 | 箱式深蹲 | 练底部爆发力 | 突破粘滞点 |

**你现在在第1-4周 做的是高杠。这个周期结束后进入低杠。每4周一换 神经系统一直有新鲜感。**

## 午间训练——V2精确计时

| 阶段 | 时间 | 内容 |
|------|------|------|
| 热身(含高翻) | 10min | 划船3min+拉伸2min+神经激活1min+高翻技术5x3 4min |
| 主项 | 12min | 卧推/深蹲 4组x120s+前面热身组 |
| 辅助项 | 13min | 3个动作 各3-4组x90s |
| 孤立+拉伸 | 10min | 手臂/侧平举/核心+拉伸5min |
| **总时长** | **45min力量+5min拉伸=50min** | 午休一小时内搞定 |

**手机计时器严格到秒——主项120s 辅助90s 孤立45-60s 超级组60s。**

## 力型兼备——热量决策标准 V2

| 看什么 | 信号 | 判断 | 调整 |
|--------|------|------|------|
| 体重趋势 | 每周涨0.25-0.5kg | ✅合适 | 继续 |
| 体重趋势 | 每周涨0.5kg以上 | ⚠️偏高 | 减200kcal |
| 体重趋势 | 不动 | ⚠️偏低 | 加200kcal |
| 体重趋势 | 下降 | ❌不够 | 加300kcal |
| 力量趋势 | 卧推/深蹲能每周+2.5-5kg | ✅合适 | 继续 |
| 力量趋势 | 连续2周不涨 | ⚠️查原因 | 先查睡眠 再加碳水 |
| 体脂趋势 | 腹肌线条还在 | ✅合适 | 继续 |
| 体脂趋势 | 线条模糊 | ⚠️偏高 | 减200kcal |
| 体脂趋势 | 更清晰但力量不涨 | ⚠️偏低 | 加200kcal |

**三个指标联动——不是只看一个。**

## 第4周减载——换动作 不是做轻一点

| 原动作 | 换成 |
|--------|------|
| 高翻(重) | 杠铃耸肩+前蹲(练分解) |
| 杠铃卧推 | 俯卧撑(3秒下+爆发推) |
| 杠铃深蹲 | 高脚杯深蹲 |
| 杠铃划船 | 哑铃单臂划船 |
| 负重引体 | 高位下拉轻量 |

**所有动作2-3组 轻重量。做完应该感觉"没练够"——这就对了。**

## 遇到情况

| 信号 | 做法 |
|------|--------|
| AMRAP比上周+2次 | ✅变强了 下次+2.5-5kg |
| AMRAP没变或下降 | ➡️维持 不退步 |
| 速度卧推速度变慢 | 减重5%——速度>重量 |
| 午间没力气 | 查11:30那顿——吃了吗 |
| 中午练完下午犯困 | 练后碳水减半碗 |
| 体重2周没涨 | 加碳水——每天多一碗饭 |
| 状态极差 | 热身+高翻+主项3组轻量+拉伸 |
